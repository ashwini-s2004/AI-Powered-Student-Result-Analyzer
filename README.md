# 🎓 Student Result Analyzer

## AI-Assisted Web Application Development Using Claude

A professional and responsive Student Result Calculator developed using HTML, CSS, and JavaScript with AI assistance from Claude.

---

## ✨ Features

* Student Name Input
* Marks Validation (0–100)
* Total Marks Calculation
* Percentage Calculation
* Grade Evaluation
* Pass/Fail Status
* Performance Remark
* Animated Progress Bar
* Grade Performance Chart
* Download Result as PDF
* Dark / Light Mode
* Confetti Celebration for Grade A
* Mobile Responsive Design

---

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
* Chart.js
* jsPDF
* Canvas Confetti
