About Project

Student Result Analyzer is a professional web application developed using AI-Assisted Web Application Development techniques with Claude. This system allows students to enter subject marks and instantly generate their Total Marks, Percentage, Grade, Performance Analysis, and Pass/Fail Status. Additional features include PDF report generation, Dark/Light mode, performance charts, and responsive design.
